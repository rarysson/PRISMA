# Petri Net

Modelador e Simulador de Rede de Petri.

## Tecnologias utilizadas

* [Vue](https://vuejs.org/)
* [JointJS](https://github.com/clientIO/joint)

## Usando o APP

Para usar localmente, clone o repositório, vá na pasta criada e execute os comandos:

```
npm install
npm run serve
```
